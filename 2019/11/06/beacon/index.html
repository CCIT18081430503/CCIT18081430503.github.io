STM32Cube Mx蓝牙功能的相关配置
一、打开STM32Cube
进入页面，在最上方的help中打开Manage embedded software packages，下载STM32F4型号1.24.1版本的固件包，然后再进入STMicroelectronics，下载X_CUBU_BLE1里的最新版本固件包（4.4.0）和X_CUBU_MEMS1的6.2.0固件包。
二、选择芯片
点击File里的New Project，进入后在左上方可搜索所需要的芯片型号，大家可根据自己的需要进行选择，我们这里用的是STM32F401RETx。
三、芯片的配置
1.点击Clock Configuration下方的Additional Software，按下图所示进行配置，设置完后点右下方的OK确定。

2.在System Core的SYS中进行配置

3.在Connectivity中配置全双工模式和串口的异步通行模式。把SPI1里的Mode的模式改为Full_Duplex Master(全双工模式)，并在其下方的Parameter Settings里配置时钟数。

在USART1的Mode里选择Asynchronous(异步模式)

4.在左下角点击Additional Software，进入STMicroelectronics.X_CUBU_BLE1.4.4.0

5.选择芯片上方的System view里的GPIO

四、进入工程
在Project Manager里设置文件名，文件位置（工程名和 工程存储路径不要用中文，不然容易出错），最后点GENERATE CODE进入工程。
